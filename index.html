ACM at CSUF presents

Illuminate the ACM Tree with your Code

Friday, December 8th
3-6pm at CS 300

Come join us for a fun-filled event where you'll get to program LEDs on our Christmas tree to create a light show together!

We'll also be decorating the tree with ornaments together, so feel free to bring your own!

Schedule

3:00pm Event Opening and Tree Decorating
3:30pm Coding Begins, SDK Introduction
5:30pm Light Show Begins

Development

For this event, we will supply you an SDK in Python and JavaScript to control the lights on the tree.

If you're feeling smart, you may also choose to interface with the tree directly using Protobuf over Websocket. The server implementation (including all definitions) can be found in diamondburned/acm-christmas.

To test out your code, we will provide you a simulator that will display LEDs on a virtual tree. You'll also be able to acquire a temporary token that lets you control the lights on the real tree for a short moment!
